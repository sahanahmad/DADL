# dadl-lab

## Project setup
```
Open Powershell:
npm install -g @vue/cli

In directory above DADL directory:
npm install vue
```

### Compiles and hot-reloads for development
```
cd DADL
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
